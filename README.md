# Smart Medicine Room - ESP32 IoT Project

## Overview
ระบบห้องเก็บยาอัจฉริยะ (Smart Medicine Storage Room)
พัฒนาโดยใช้ ESP32 + Web Dashboard + MySQL + Telegram Notification

## Hardware
- ESP32
- DHT22
- IR Sensor
- Relay
- Servo Motor (Door)
- Buzzer
- LED Status

## Features
- Real-time Monitoring
- Web Control (Relay, Door, Fan)
- Telegram Notification
- Edge Automation (ทำงานแม้เน็ตหลุด)

## How to Setup
1. Flash firmware ไปยัง ESP32
2. เชื่อม WiFi ผ่าน IOT-SETUP
3. ตั้งค่า API / Database
4. เปิด Dashboard
